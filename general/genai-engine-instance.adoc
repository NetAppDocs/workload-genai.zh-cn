= NetApp GenAI引擎的组件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
当您部署 GenAI 基础设施时，Workload Factory 会为 GenAI 引擎创建一个 EC2 实例。它还为此实例创建 IAM 角色、安全组和私有端点。您可能希望了解有关 Workload Factory 在您的 AWS 环境中创建的这些组件的更多详细信息。

EC2实例类型:: m5.large
IAM角色:: GenAI引擎实例需要具有向Amazon Brock上的嵌入模型发送数据区块以及与NetApp AI服务后端进行通信的权限。IAM角色包括以下权限：

安全组:: 出站规则对所有流量开放、而入站规则完全关闭。
专用端点:: 如果目标 VPC 还没有它们，Workload Factory 会为 GenAI 引擎 EC2 实例创建私有端点，以便它可以与以下 AWS 服务通信：

+

* 适用于 NetApp ONTAP 的 Amazon FSX
